Este repositorio contiene notebooks (.ipynb) y materiales desarrollados durante el curso de Minería de Datos. Incluye tanto conceptos teóricos clave como aplicaciones prácticas en Python para el análisis y exploración de datos.

El objetivo es consolidar el aprendizaje de temas como minería de datos, machine learning, análisis exploratorio (EDA) y metodologías como CRISP-DM, mediante ejemplos prácticos y resúmenes estructurados.

Contenido principal:
- Notebooks en Python (análisis y práctica)
- Resúmenes teóricos
-Ejercicios y casos aplicados

Este repositorio sirve como guía de estudio y referencia para comprender cómo transformar datos en conocimiento útil.
